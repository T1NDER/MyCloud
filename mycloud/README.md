# My Cloud — Облачное хранилище файлов

**My Cloud** — это полнофункциональное веб-приложение облачного хранилища файлов, разработанное в рамках дипломного проекта по профессии «Fullstack-разработчик на Python».

Приложение позволяет пользователям регистрироваться, загружать файлы в личное хранилище, управлять ими (скачивать, переименовывать, удалять), добавлять комментарии и делиться файлами через специальные ссылки. Администраторы могут управлять пользователями и просматривать хранилища любых пользователей.

---

## Возможности

### Для пользователей:
- ✅ Регистрация с валидацией данных (логин, email, пароль)
- ✅ Аутентификация и выход из системы
- ✅ Загрузка файлов с добавлением комментария
- ✅ Просмотр списка загруженных файлов
- ✅ Скачивание файлов с сохранением оригинального имени
- ✅ Переименование файлов
- ✅ Удаление файлов
- ✅ Копирование специальной ссылки для внешнего доступа
- ✅ Скачивание файла по специальной ссылке (без регистрации)

### Для администраторов:
- ✅ Просмотр списка всех пользователей
- ✅ Информация о хранилищах пользователей (количество файлов, общий размер)
- ✅ Удаление пользователей
- ✅ Назначение/снятие прав администратора
- ✅ Просмотр файлов любого пользователя
- ✅ Управление файлами от имени любого пользователя

##  Установка и запуск

### 1. Клонирование репозитория

```bash
git clone https://github.com/ВАШ_ЛОГИН/mycloud.git
cd mycloud
```

### 2. Запуск backend

### Переход в папку backend
cd backend

### Создание виртуального окружения
python -m venv venv

### Активация виртуального окружения
### Windows:
venv\Scripts\activate
### Linux/macOS:
source venv/bin/activate

### Установка зависимостей
pip install --upgrade pip
pip install -r requirements.txt

### Применение миграций
python manage.py migrate

### Создание суперпользователя (администратора)
python manage.py createsuperuser

### Запуск сервера разработки
python manage.py runserver

Backend будет доступен по адресу: http://localhost:8000
Django Admin: http://localhost:8000/admin/
   
## 3. Запуск Frontend

Откройте новый терминал (не закрывая backend):

# Переход в папку frontend
cd frontend

# Установка зависимостей
npm install

# Запуск сервера разработки
npm run dev

Frontend будет доступен по адресу: http://localhost:5173

### 4. Учетные данные и Валидация

Создание администратора
После запуска миграций создайте суперпользователя:

```
bash

python manage.py createsuperuser
```

Введите:
- Логин: admin (или любой другой)
- Email: admin@example.com
- Пароль: должен соответствовать требованиям (см. ниже)

Правила валидации данных

Логин:
- Только латинские буквы и цифры
- Первый символ — буква
- Длина от 4 до 20 символов

Пример: testuser123

Email:
- Стандартный формат email
- Уникальный в системе

Пример: user@example.com

Пароль:
- Минимум 6 символов
- Минимум одна заглавная буква (A-Z)
- Минимум одна цифра (0-9)
- Минимум один специальный символ (@$!%*?&)

Пример: Password123!

Сборка для production
Backend:

```
bash

cd backend
python manage.py collectstatic --noinput
```

Frontend:
```
bash
cd frontend
npm run build
```
